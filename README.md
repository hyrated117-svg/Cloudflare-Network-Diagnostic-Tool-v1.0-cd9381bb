# 🌐 Cloudflare Network Diagnostic Tool  
### SwiftUI App • MCP Server • AppMaker Generator • Multi‑Platform Installer Pack

A fully modular, SwiftUI‑powered diagnostic suite for Cloudflare networking, paired with a Python/Node MCP server and an AppMaker engine that generates entire SwiftUI applications on demand.

This project includes:

- A **SwiftUI diagnostic app** (DNS, DoH, Trace, WARP, Resolver Benchmark, Health Score)
- A **Python + Node MCP server** for automation and tool execution
- A **Swift‑based AppMaker generator** that builds full SwiftUI projects
- A **multi‑platform installer pack** (macOS, Linux, Windows, NPM)
- A complete **developer documentation suite**

---

## 🚀 Features

- **Cloudflare diagnostic engine**  
  DNS, DoH, Trace, WARP, resolver benchmarking, latency scoring, and more.

- **SwiftUI interface**  
  Clean, modular, and fully theme‑aware.

- **MCP server integration**  
  Python + Node servers exposing diagnostics, export tools, and project generation.

- **AppMaker generator**  
  Generates SwiftUI apps, injects modules, writes files, and compresses output.

- **Cross‑platform installers**  
  macOS/Linux shell installer, Windows PowerShell installer, Python installer, and NPM global installer.

- **Developer‑friendly architecture**  
  Clear folder structure, templates, docs, and tools.
